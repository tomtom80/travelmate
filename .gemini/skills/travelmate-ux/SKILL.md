# Travelmate UX Design Skill

Expertise in UX design and interaction patterns for the Travelmate project.

## Core Mandates
- Tech Stack: Thymeleaf + HTMX + PicoCSS 2.
- Responsive design: Desktop, Tablet, Mobile.
- Accessibility: ARIA labels, semantic HTML.

## Specialized Workflows

### 🛤️ 1. User Journey Mapping
- Map flows: Sign-up, Trip creation, Invitation, Expense tracking.
- Phases: Discovery, Onboarding, Core Task, Completion.
- **Reference**: See [ux-design-specs.md](references/ux-design-specs.md).

### 📐 2. Figma Design Specifications
- Component Specs (PicoCSS base).
- Variants: Default, Hover, Error, Loading (HTMX).
- Layout & responsive behavior.
- **Reference**: See [ux-design-specs.md](references/ux-design-specs.md).

### 🧩 3. Design System (PicoCSS + HTMX)
- Forms, Buttons, Cards, Navigation.
- HTMX Interaction Patterns: `hx-get`, `hx-post`, `hx-target`, `hx-swap`.
- Loading indicators: `hx-indicator`.
- **Reference**: See [ux-design-specs.md](references/ux-design-specs.md).

### 🌍 4. i18n Design
- German and English support (`th:text="#{key}"`).
- Text expansion considerations.
- Ubiquitous language application (ADR-0011).
