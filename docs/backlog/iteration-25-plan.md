# Iteration 25 — Major Release

**Target Version**: `v1.0.0`  
**Status**: PLANNED

## Goal

Iteration 25 is the release iteration for the first production-ready major version.

- final regression and release stabilization
- load-test baseline and documented operational limits
- rollback drill and release smoke checks
- final docs review across backlog, arc42, operations, and release notes
- release candidate sign-off and final tag preparation

## Planned Deliverables

- documented and verified major-release decision package
- reproducible release and rollback path
- `v1.0.0` release notes and release checklist

## Acceptance

- go-live gates from [`roadmap-v1.0.0.md`](./roadmap-v1.0.0.md) are satisfied or formally waived with explicit risk ownership
- release smoke tests pass on the target deployment path
- final release artefacts are complete and internally consistent
